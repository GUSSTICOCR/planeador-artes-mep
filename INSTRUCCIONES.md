# 🎨 Guía de Uso: Planeador de Artes Plásticas IA

¡Felicidades! Tu aplicación está lista. Sigue estos pasos para usarla al 100%:

## 🚀 Cómo Iniciar la App
1.  **Node.js**: Asegúrate de tener Node.js instalado en tu computadora.
2.  **Un Solo Clic**: Busca el archivo llamado `INICIAR_APP.bat` en esta carpeta y dale doble clic.
3.  **Automático**: Se abrirá una ventana negra (el servidor) y automáticamente tu navegador abrirá la dirección: `http://localhost:3000/galeria.html`.

> [!IMPORTANT]
> **No cierres la ventana negra** mientras estés trabajando con la aplicación, ya que ese es el "cerebro" que conecta con Gemini y genera los PDFs.

## 🔑 Configuración de la API Key
Para que la IA funcione, necesitas tu Clave API de Gemini:
1.  Abre la aplicación.
2.  Haz clic en el botón grande **"Generador de Planeamientos IA"** (el que tiene el logo de Gemini).
3.  En el formulario de la izquierda, pega tu clave en el campo que dice **"Clave API de Gemini"**.
4.  ¡Listo! Ya puedes rellenar el tema y nivel para generar tus planeamientos oficiales del MEP.

## 🖼️ Exploración e Investigación
-   Usa la **Galería** para ver los periodos artísticos.
-   Haz clic en cualquier tarjeta para ver el **Resumen e Imágenes**.
-   Si necesitas más info, usa el botón **"✨ Consultar a Gemini"** para que la IA te responda en el chat lateral o **"🌐 Ver en la Web"** para buscar en Google.
